# Cookie Clicker

This is a static site, so the easiest way to put it online is Vercel.

## Deploy To Vercel

1. Create a new GitHub repository.
2. Upload these files to the repo root:
   - `index.html`
   - `style.css`
   - `script.js`
   - `manifest.webmanifest`
   - `service-worker.js`
   - `icon.svg`
3. Go to Vercel and sign in with GitHub.
4. Import the repository.
5. Keep the defaults and deploy.
6. Wait for Vercel to publish the site.

## Open On Phone

1. Open the Vercel URL on your phone.
2. On Android, use the browser menu and choose Install app or Add to Home screen.
3. On iPhone, use Share then Add to Home Screen.

## Notes

- The manifest and service worker make the site more installable and give it offline caching once it is hosted.
- If you want, I can also help you deploy it directly from the command line with the Vercel CLI.
